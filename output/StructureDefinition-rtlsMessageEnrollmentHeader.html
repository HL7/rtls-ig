Resource Profile: RTLS Message Enrollment Header Resource Profile ( Experimental )

Official URL: http://hl7.org/fhir/uv/rtls/StructureDefinition/rtlsMessageEnrollmentHeader Version: 1.0.0-ballot2
Standards status: Informative Computable Name: RTLSMessageEnrollmentHeader

MessageHeader resource for FHIR enrollment messages sent in RTLS workflows.

This profile defines minimum expectations for the MessageHeader resource to be used in RTLS enrollment workflows.

Usages:

  • Use this Profile: RTLS Enrollment Message Profile
